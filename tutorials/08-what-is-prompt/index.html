定义#

Prompt = 提示词 = 你对 AI 说的话

Prompt 就是你输入给 AI 的文字，AI 根据你输入的内容来生成回答。

为什么 Prompt 重要？#

AI 的回答质量，很大程度上取决于你问的问题。

问法AI 回答
“写一篇关于狗的文章”可能很泛泛
“写一篇 500 字的柯基犬饲养指南，面向新手”精准满足需求

Prompt 的组成#

一个好的 Prompt 可以包含：

1. 任务描述#

要 AI 做什么

2. 背景信息#

为什么做这件事

3. 具体要求#

格式、长度、风格等

4. 输出格式#

表格？列表？纯文本？

A#

API (Application Programming Interface)#

应用程序接口

  • 简单说就是 AI 程序给外界用的"电话"
  • 你打电话给 AI，AI 通过 API 回答你
  • 用 API 可以让程序自动调用 AI

AI (Artificial Intelligence)#

人工智能

  • 让计算机像人一样思考和做事

AGI (Artificial General Intelligence)#

通用人工智能

  • 能完成人类所有智能任务的 AI
  • 目前还不存在

B#

Bot#

机器人

  • 自动执行的程序
  • 如：聊天机器人、投票机器人

C#

ChatGPT#

OpenAI 开发的聊天 AI

  • 2022年11月发布
  • 目前最流行的 AI 聊天工具

Claude#

Anthropic 公司开发的 AI

  • 以安全著称
  • 特别擅长写作和分析

Context Window#

上下文窗口

  • AI 能记住多少对话内容
  • 越长越好

D#

Dataset#

数据集

  • 训练 AI 用的大量数据

Deep Learning#

深度学习

  • 使用多层神经网络的机器学习

E#

Embedding#

向量嵌入

  • 把文字转换成数字的技术
  • 让计算机能理解文字的"意思"

F#

Fine-tuning#

微调

  • 在已有模型基础上继续训练
  • 让 AI 更擅长某个特定任务

G#

GPT#

Generative Pre-trained Transformer

  • OpenAI 的大语言模型系列
  • GPT-3、GPT-4、GPT-4o 等

H#

Hallucination#

幻觉

  • AI 胡说八道
  • 编造不存在的事实

L#

LLM (Large Language Model)#

大语言模型

  • 训练了大量文本的 AI
  • 能理解也能生成文字

M#

Machine Learning#

机器学习

  • 让计算机从数据中学习

Model#

模型

  • AI 的大脑/程序
  • 经过训练后能完成特定任务

N#

Neural Network#

神经网络

  • 模仿人脑的计算模型

P#

Prompt#

提示词

  • 你对 AI 说的话
  • AI 根据 Prompt 生成回答

Prompt Engineering#

提示词工程

  • 研究如何写更好的 Prompt

R#

RAG (Retrieval Augmented Generation)#

检索增强生成

  • 让 AI 能查阅外部资料
  • 减少 AI 胡说八道

S#

Streaming#

流式输出

  • AI 边想边说
  • 不用等完整答案

T#

Token#

令牌/词元

  • AI 处理文字的最小单位
  • 约等于半个中文词或一个英文单词

Transformer#

Transformer 架构

  • 现在大语言模型的核心技术

常见词#

术语解释
训练让 AI 学习数据
推理AI 使用学到的知识
输出AI 生成的回答
参数AI 模型的"脑容量"
温度控制 AI 回答的随机性

通俗解释#

API = 应用程序接口

想象一下：

  • 你是餐厅顾客（你的程序）
  • 厨师是 AI 服务商
  • 服务员就是 API

你不需要知道厨房怎么做菜，只需要告诉服务员你要什么，服务员会把菜端给你。

为什么要用 API？#

不用 API#

  • 只能网页上手动操作
  • 每次都要复制粘贴
  • 效率低

用 API#

  • 程序自动调用
  • 可以批量处理
  • 效率高

AI API 能做什么？#

有了 AI API，你可以：

  1. 批量生成内容

    • 自动写 100 篇文章
    • 批量生成产品描述

  2. 做自己的 AI 产品

    • 做聊天机器人
    • 做 AI 助手

  3. 自动化工作

    • 自动回复邮件
    • 自动处理数据

怎么使用 AI API？#

第一步：获取 API Key#

  1. 访问 AI 服务商官网
  2. 注册账号
  3. 获取 API Key（类似密码）

import requests

# 简单的 API 调用示例
response = requests.post(
    "https://api.openai.com/v1/chat/completions",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "model": "gpt-4",
        "messages": [{"role": "user", "content": "你好"}]
    }
)

print(response.json())

## 🚀 快速开始

### 只需要 3 步，立刻用上 AI

1. **注册账号** → 访问 [chat.openai.com](https://chat.openai.com) 注册
2. **开始对话** → 像聊天一样告诉 AI 你的需求
3. **复制结果** → AI 生成的内容可以直接复制使用

### 常见使用场景

| 场景 | 示例 Prompt |
|------|------------|
| 写文章 | "帮我写一篇关于 AI 发展的文章" |
| 写代码 | "用 Python 写一个计算器" |
| 解释概念 | "什么是机器学习？" |
| 翻译 | "把这段话翻译成英文" |
| 总结 | "帮我总结这篇文章的主要内容" |

## ❓ 常见问题

**Q: AI 是什么？**
A: AI 即人工智能（Artificial Intelligence），简单说就是让计算机像人一样思考和做事。

**Q: 使用 AI 需要付费吗？**
A: 基本使用免费，付费版功能更强。ChatGPT 免费版够用，Claude 免费版性价比高。

**Q: AI 会取代人类工作吗？**
A: AI 是工具，不会取代人类，但会使用 AI 的人会取代不会用 AI 的人。

**Q: 英语不好能用 AI 吗？**
A: 能！中文 AI 也很强，用中文提问就行。

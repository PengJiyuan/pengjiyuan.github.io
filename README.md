## 🚀 快速开始

### 3 步上手 AI

```bash
# 1. 注册账号
访问 https://chat.openai.com 注册

# 2. 开始对话
像聊天一样告诉 AI 你的需求

# 3. 复制结果
AI 生成的内容可以直接使用
```

### 常见场景 Prompt

| 场景 | Prompt 示例 |
|------|-------------|
| 写文章 | "帮我写一篇关于 AI 发展的文章" |
| 写代码 | "用 Python 写一个计算器" |
| 解释概念 | "什么是机器学习？" |
| 翻译 | "把这段话翻译成英文" |

## ❓ 常见问题

**Q: AI 是什么？**
> AI 即人工智能，让计算机像人一样思考和做事。

**Q: 使用 AI 需要付费吗？**
> 基本使用免费，付费版功能更强。ChatGPT/Claude 免费版都够用。

**Q: 英语不好能用 AI 吗？**
> 能！直接用中文提问就行。

**Q: AI 会取代人类工作吗？**
> AI 是工具，不会取代人类。但会使用 AI 的人会取代不会用 AI 的人。

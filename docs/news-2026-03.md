# 📰 AI 资讯速递 (2026年3月)

> 每日AI行业动态和技术资讯

---

## 🚀 本周热点 (2026年3月 第1周)

### 大模型进展

1. **Claude Code 语音模式正式发布 (2026.3)**
   - 输入 `/voice` 开启语音编程，长按空格说话
   - 语音转录 Token 完全免费
   - 编程进入「对讲机」时代，交互方式革命

2. **OpenAI Codex 语音编程功能更新**
   - 几乎同一时间推出类似功能
   - 支持 macOS 和 Windows

3. **GPT-5.4 意外泄露**
   - OpenAI 最新模型瞄准两大能力突围
   - 多处踪迹被扒，引发社区关注

4. **GPT-5 更新至 5.3-Codex (2026.2)**
   - 最强大、全面、Agent 能力增强
   - 推荐场景：复杂任务、编程

5. **Claude Opus/Sonnet 4.6 (2026.2)**
   - 安全、写作强、长任务处理能力提升
   - 推荐场景：长文本、分析、编程

6. **谷歌 Gemini 升级至 3.1 Pro 版本**
   - 多模态能力进一步提升

7. **通义千问 Qwen 2.5 发布**
   - 中文能力强大，部分版本免费开放

---

### 行业动态

1. **OpenClaw 登顶 GitHub 星标榜 (2026.3)**
   - 仅用两个月超越 Linux，成为史上最受欢迎开源项目
   - 由奥地利独立开发者推出，本地运行 AI Agent 框架
   - 接入 WhatsApp、Telegram、Discord 等通讯工具
   - 引发「注意力经济」讨论，代表当下技术情绪

2. **Claude 全球宕机事件 (2026.3)**
   - 机房爆炸，美财政部全面停用
   - ChatGPT 卸载量狂飙 295%

3. **OpenAI 与五角大楼达成模型部署协议**
   - 国防合同引发关注
   - AI 在国防领域的应用引起讨论

4. **Anthropic 收购视觉 AI 创企 Vercept**
   - 继续强化多模态能力

5. **Anthropic 发布「经济原语」衡量 AI 影响**
   - 从经济学角度评估 AI 对劳动力市场的影响

6. **银河通用机器人完成 25 亿元融资**
   - 具身智能领域最大规模融资之一

7. **松延动力 B 轮融资近 10 亿元**
   - 具身智能公司融资火热

---

### AI 应用与市场

1. **QuestMobile 2025 AI 应用报告**
   - 移动端 AI 月活达 7.22 亿
   - 原生 App 巨头筑墙，新锐破局
   - AI 核心入口雏形初现

2. **人形机器人融资热潮**
   - 融资 100 亿，量产竞赛加速
   - 2028 年被视为人形机器人「生死线」

3. **前小米高管创业机器人**
   - 小雨智造用「爆品逻辑」做工业通用具身智能

---

### 技术趋势

- **MCP (Model Context Protocol)** 成为 AI 应用开发新标准
- **AI Agent** 自动完成任务的能力显著提升
- **语音编程** 成为新战场，交互方式革命
- **RAG** 技术在企业知识库领域广泛应用
- **端侧 AI** 发展迅速，移动端部署成为热点

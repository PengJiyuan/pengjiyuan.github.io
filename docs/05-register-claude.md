# 如何注册 Claude？

## 什么是 Claude？

**Claude = Anthropic 公司开发的 AI 助手**

- 以安全、可靠著称
- 特别擅长写作、分析
- 免费版够用

---

## 注册步骤

### 第一步：访问官网

打开 **claude.ai**

### 第二步：注册

1. 点击 "Start building with Claude"
2. 用邮箱注册 或 Google 账号登录
3. 填写信息

### 第三步：开始使用！

注册完就能用了，直接在对话框输入问题即可。

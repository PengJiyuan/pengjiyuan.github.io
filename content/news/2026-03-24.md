## 🤖 技术突破

### Anthropic发布Claude Code Channels：Discord/Telegram直控AI编程代理

Anthropic今日发布**Claude Code Channels**，将其强大的Claude Code AI Agent接入Discord和Telegram，用户随时随地可通过消息应用向Claude Code下达编程指令。这一更新被业界视为"OpenClaw杀手"——用户不再需要购买Mac Mini等专用硬件来7×24小时运行开源Agent，Anthropic将持久化运行环境直接集成进Claude Code生态。AI YouTuber Matthew Berman评价："他们把OpenClaw给造出来了。"

Claude Code Channels支持用户将Agentic AI工作流延伸至日常聊天场景，同时保持Claude Code原有的代码编写、执行、工具调用能力。这标志着AI编程Agent从开发者工具向消费级产品的重大跨越。

来源：[VentureBeat](https://venturebeat.com/orchestration/anthropic-just-shipped-an-openclaw-killer-called-claude-code-channels/)

### 英伟达开源Nemotron-Cascade 2：3B参数斩获数学编程三金牌

英伟达发布**Nemotron-Cascade 2**，这是一个300亿参数的MoE模型，仅使用30亿活跃参数即可运行，却在数学和编程基准测试中斩获金牌级表现。具体成就：

- **2025年国际数学奥林匹克（IMO）金牌**
- **2025年国际信息学奥林匹克（IOI）金牌**
- **2025年ACM-ICPC全球总决赛金牌**

这是继DeepSeek-V3.2-Speciale之后，第二个达成"IMO+IOI+ICPC三冠王"的开放权重模型。更重要的是，Nemotron-Cascade 2的Cascade RL后训练Pipeline、训练数据集和强化学习配方全部开源，并采用宽松许可证——对需要构建领域特定推理系统的企业团队而言，这是目前最详细的开放后训练方法论。英伟达还宣布成立**Nemotron Coalition**，Mistral作为联合开发方加入，共同打造该联盟首款开放前沿基础模型。

来源：[VentureBeat](https://venturebeat.com/orchestration/nvidias-nemotron-cascade-2-wins-math-and-coding-gold-medals-with-3b-active/)

### Luma AI发布Uni-1：图像生成超越Google，成本降低30%

Luma AI于3月22日发布**Uni-1**图像生成模型，直接挑战Google在AI图像生成领域的主导地位。Uni-1不仅在图像质量评分上超越Google Nano Banana家族模型，生成成本还**降低最多30%**——这对追求规模化生产的高分辨率图像团队而言具有决定性意义。

Luma AI此前以Dream Machine视频生成工具闻名，此番进入图像生成赛道，凭借"更便宜+更高质量"的组合拳重新定义了行业竞争格局。150人团队挑战科技巨头的叙事，令整个行业为之侧目。

来源：[VentureBeat](https://venturebeat.com/technology/luma-ai-launches-uni-1-a-model-that-outscores-google-and-openai-while/)

### Mistral Small 4：推理+视觉+编码三合一，Apache 2.0开源

Mistral AI发布**Small 4**，这是一款将推理、视觉和编码能力整合进单一Apache 2.0开源模型的紧凑型模型。Small 4仅需2台DGX B200即可运行，在LCR基准测试中击败GPT-OSS 120B，但在推理基准上仍落后于Qwen和Haiku。

**核心优势：**
- 更短输出=更低延迟+更低Token成本
- 与Nvidia深度合作，推理已针对开源vLLM和SGLang优化
- 指令遵循性能优异，适合高吞吐量企业任务（如文档理解）
- 已在今年3个月内密集发布Small 4、Leanstral（形式验证代码Agent）、Forge（企业私有模型构建平台），并加入英伟达Nemotron Coalition

来源：[VentureBeat](https://venturebeat.com/technology/mistrals-small-4-consolidates-reasoning-vision-and-coding-into-one-model-at/)

### Cursor Composer 2：基于Kimi K2.5的深层问题

VentureBeat深度报道披露，Cursor爆款编程工具Composer 2的背后，是**月之暗面的Kimi K2.5**（1万亿参数、320亿激活参数）而非任何西方基础模型。这一发现揭示了一个尴尬现实：在西方开源模型青黄不接的当下，Cursor等明星AI产品公司被迫转向中国模型。

文章指出更深层问题：Meta的Llama 4 Behemoth（2万亿参数）无限期延迟，Google的Gemma 4尚未发布，西方在开源前沿模型的竞争中明显落后。Cursor表示下次会披露基础模型，但真正的问题是：下次他们能否拿出可信的西方替代方案？

来源：[VentureBeat](https://venturebeat.com/technology/cursors-composer-2-was-secretly-built-on-a-chinese-ai-model-and-it-exposes-a/)

### MiniMax M2.7：可完成30-50%强化学习研究工作流

VentureBeat报道，MiniMax发布**M2.7**专有AI模型，具备"自我进化"能力——可自主执行30-50%的强化学习研究工作流环节。这是继OpenAI o1/o3、DeepSeek R1之后，AI模型在科研自动化领域的又一突破。

来源：[VentureBeat](https://venturebeat.com/)

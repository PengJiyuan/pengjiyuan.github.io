AI 资讯速递（2026年3月7日）

2026年3月7日：特朗普封杀Anthropic、OpenAI发布GPT-5.4、Claude Code新技能、智能经济新形态首入政府工作报告。

政策与监管#

  1. 特朗普下令联邦政府停用Anthropic技术

    • 美国总统特朗普指示所有联邦机构立即停止使用Anthropic技术
    • 战争部等机构有6个月逐步淘汰期
    • 特朗普警告Anthropic若不配合将面临民事和刑事后果
    • 此前Anthropic与五角大楼在军事AI应用方面存在分歧，拒绝将AI模型用于大规模国内监控和自主杀人

  2. 智能经济首次写入政府工作报告

    • “打造智能经济新形态"被首次写入政府工作报告
    • 从"工具赋能"到"范式重构”，AI成为生产力新核心
    • 专家：将对生产链、供应链乃至价值链从最上游到末端产生重构
    • “算电协同"首次被写入政府工作报告

  3. 印尼对Meta发出严厉警告

    • 印尼通信与数字部指责Meta在网络赌博和虚假信息传播方面管控不力
    • 仅对约28.47%被标记的涉及网络赌博和虚假信息的内容进行了整改
    • 政府敦促Meta立即加强内容审核机制

  4. 全国人大代表陈达：AI替代从"体力"向"脑力"延伸

    • AI不再局限于替代生产线上的重复性体力劳动
    • 更通过大模型、生成式AI等工具，开始介入内容创作、代码编写、数据分析等岗位

技术前沿#

  1. OpenAI发布GPT-5.4：原生支持计算机操控

    • OpenAI正式推出最强模型GPT-5.4
    • 原生支持计算机操控功能，标志着AI在自主操作系统方面取得重大突破
    • 被评价为"可能是世界上最好的模型”

  2. 微软开源Phi-4视觉推理模型

    • 微软发布Phi-4-Reasoning-Vision-15B
    • 专注于视觉推理能力的15B参数模型

  3. Claude Code推出/simplify和/batch技能

    • Anthropic即将在Claude Code中引入两个新技能
    • /simplify和/batch可自动化代码审查和并行代码迁移工作

  4. PyTorch生态新增Kubetorch工具

    • 支持在Kubernetes上进行ML研发
    • 涵盖训练、推理、强化学习、评估和数据处理等全流程

  5. 脑机接口首次写入政府工作报告

    • 政府工作报告提出，培育发展未来能源、量子科技、具身智能、脑机接口、6G等未来产业
    • 脑机接口正加快走出实验室，政策东风下发展前景值得期待

行业应用#

  1. 北京"机器人4S店"开进汽车4S店

    • 京城汽车4S店开卖机器人
    • 修车保养间隙可和机器人下五子棋
    • “机器人4S店+汽车4S店"的跨界组合

  2. 天津港全国人大代表展示智能机器人"徒弟”

    • 全国人大代表成卫东介绍天津港智能运输机器人
    • 2021年首个"智慧零碳"码头投产运营，智能运输机器人代替老式拖车

  3. 上海"803反诈智能体"升级

    • 面向全国，用户可将可疑电话或短信一键输入
    • 系统可判断是否为诈骗并给出应对建议

  4. AI浏览器成新战场

    • Perplexity推出Comet AI浏览器，OpenAI发布ChatGPT Atlas
    • 谷歌Chrome和微软Edge分别集成Gemini和Copilot
    • AI浏览器何时安全成为关注焦点

资本与市场#

  1. Anthropic年化营收逼近200亿美元

    • 面向企业的专注策略令投资者对其上市充满期待
    • 预计2026年底IPO，估值或超5000亿美元

  2. VAST完成5000万美元A轮融资

    • 将加速构建世界模型与UGC互动内容平台
    • 推动AI生成内容和用户创作内容的深度融合

  3. 甲骨文和OpenAI放弃德州数据中心扩建计划

    • 双方在融资问题以及OpenAI不断变化的需求上谈判拖延过久
    • Meta或接手，英伟达介入促成Meta与开发商讨论

  4. 佰维存储冲刺千亿市值

    • 连续三日大涨，涨幅超42%
    • 前两个月净利润预计达15-18亿元

  5. 液冷技术迎规模化放量元年

    • 2026年预计700-800亿元市场规模
    • AI数据中心液冷渗透率将升至40%

企业动态#

  1. OpenClaw登顶GitHub趋势榜

    • “养虾人"从硅谷蔓延至中国
    • 腾讯免费帮安装OpenClaw活动吸引数百用户部署

  2. Anthropic推出AI应用市场

    • “Anthropic Marketplace"对标亚马逊商店
    • 允许客户购买第三方工具，不抽取佣金

  3. OpenAI和Meta研究员转用Claude Max

    • 多位OpenAI和Meta研究人员升级订阅Claude Max 20x计划
    • 显示出Anthropic产品在AI研究社区中的吸引力

  4. 阿里千问"一句话办事"完成2亿订单

    • 超400万60岁以上用户首次使用AI点外卖、打车
    • AI办事成AI眼镜刚需，无屏化操作成趋势

专家观点#

  1. 麦肯锡：善用AI的毕业生就业更吃香

    • 受AI和大语言模型影响最大的行业招聘岗位数量下降38%
    • 但善于驾驭AI工具的人才正成为就业市场青睐对象

  2. Anthropic研究报告：先拥抱AI的行业或许先被AI吃掉

    • AI系统性"去技能化”：抽走高智力、高判断、高创造部分
    • 留下低价值执行任务，岗位核心价值被掏空

  3. 专家解读"智能经济新形态”

    • 北京师范大学教授万喆：底层逻辑从"工具赋能"到"范式重构"
    • 重新对生产链、供应链乃至价值链产生重构

今日小结#

2026年3月7日的AI行业呈现以下核心事件：特朗普政府下令联邦机构停用Anthropic技术，标志着AI伦理与国家安全博弈升级；OpenAI发布GPT-5.4原生支持计算机操控，AI从"聊天"向"做事"进化加速；“智能经济新形态"首次写入政府工作报告，AI发展范式迎来重构；具身智能、脑机接口等未来产业获得政策明确支持。两会期间，AI成为代表委员热议焦点，从顶层设计到产业落地，中国AI发展路径日益清晰。

政策与治理#

  1. 全国两会代表委员热议AI赋能

    • 构建"技术突破—场景验证—产业应用"发展闭环
    • 加速具身智能技术应用场景落地
    • 鼓励专用推理芯片国产化，重点突破高精度、低时延、多模态芯片技术
    • 支持智能体技术深度应用
    • 构建行业通用大模型并建立统一规范与标准体系

  2. 王江平：AI治理需聚焦具体行业

    • 全国政协委员、工业和信息化部原副部长王江平强调AI治理需"双向对齐"
    • 制造业AI需满足高鲁棒性、可解释性等底层要求
    • 呼吁全球协同治理

  3. 周鸿祎：支持中小企业低成本构建垂直领域智能体

    • 建议支持打造安全智能体产品，在重点领域批量部署
    • 推动场景化应用
    • 关注推理算力布局优化和智能体公共服务平台建设

  4. 张云泉：建议将终端智能体纳入监管

    • 建议规范数据采集和使用边界
    • 禁止无差别读取手机、电脑等屏幕内容
    • 建议补贴消费者购买智能体订阅服务

  5. 越南成为东南亚首个出台AI专门立法国家

    • 《人工智能法》于2026年3月1日正式生效
    • 以风险导向治理为主线，强调高风险系统合规评估
    • 在治理逻辑上明显向欧盟《人工智能法》靠拢

  6. 武汉启动"AI+教育"创新应用征集

    • 面向全市征集智能备课、个性化学习等应用场景
    • 教育领域大模型、AI教师助手等智能体
    • 教学机器人、VR/AR学习终端等智能硬件产品
    • 今年武汉将推进"人工智能行业应用年"，教育领域首个重点推进

技术前沿#

  1. IBM：2026年将标志着量子计算机首次超越传统计算机

    • 量子计算机解决问题的能力将优于所有仅使用经典方法的计算机
    • 为药物研发、材料科学、金融优化等行业开启新突破之门

  2. 2026十大AI趋势发布

    • AI从"聊天"转向"做事"的智能体时代
    • 开源模型与Agent协同的标准逐渐形成
    • “All in One"AI应用将更普遍化
    • AI+机器人、AI+工业软件成关键赛道

  3. Yann LeCun计划离开Meta投身世界模型创业

    • 世界模型是能理解现实世界动态的神经网络
    • 包括物理法则与空间特性，可生成仿真真实物理环境的视频
    • 物理AI开发者使用世界模型来产生定制化合成数据

  4. Zoom领导力洞察：2026年AI关键趋势

    • 联邦式AI和智能体AI的崛起
    • 营销中的真实性、权威性和敏捷性
    • 客户体验的演进
    • 信任、透明度和监管

  5. 中国具身智能模型获全球统一标准第一

    • 2026年1月，中国具身智能模型在全球统一标准下获得第一
    • 意味着中国团队训练出的机器人"大脑"具备了在物理世界理解和执行任务的能力

资本与市场#

    • 在韩国新万金地区投资约9万亿韩元
    • 建设涵盖AI数据中心、机器人及氢能产业的综合性项目

  2. 中国人工智能产业发展联盟发布"大模型一体机产业图谱”

    • 收录70余家代表性企业
    • 全面展示大模型一体机产业生态体系
    • 助力AI普惠化发展

  3. 高盛：中国AI走出独立行情

    • 认为中国AI潜在经济效益远未被充分定价
    • 自DeepSeek时刻以来，中国AI相关股票平均上涨约50%
    • 相关科技股总市值新增逾3万亿美元

  1. 阿里Qwen3-Max-Thinking模型发布

  2. 月之暗面开源Kimi K2.5模型

  3. 谷歌DeepMind开放Genie 3工具

    • 用户通过自然语言描述可即时创建并探索可交互的三维虚拟世界
    • AI在模拟现实世界的前沿研究方面又迈进一步

  4. 英伟达推出Vera Rubin系统级AI算力平台

    • 包含6款芯片的系统级AI算力平台
    • 微软发布AI芯片Maia 200

  1. AI"上天"：中国Qwen3大模型部署至太空计算中心

  2. NASA毅力号火星车完成AI规划路线行驶

    • 2026年1月30日首次完成由AI规划路线的火星表面行驶任务

  3. AlphaGenome解锁人类"暗基因组"

    • 英国《自然》期刊封面文章介绍深度学习模型AlphaGenome成果
    • 能解码人类基因组中占98%的"暗基因组"
    • 将有助于深入了解遗传疾病、改进基因检测、研发新药物和疗法

  4. OpenClaw等智能体改变工作方式

    • 智能体不仅会聊天，还能操作电脑、应用程序和网络完成复杂任务
    • 据报道已有上百万个智能体在Moltbook平台上交互

大模型进展#

  1. Claude Code 语音模式正式发布

    • 输入 /voice 开启语音编程，长按空格说话
    • 语音转录 Token 完全免费
    • 编程进入「对讲机」时代，交互方式革命

  2. OpenAI Codex 语音编程功能更新

    • 几乎同一时间推出类似功能
    • 支持 macOS 和 Windows

  3. GPT-5.4 意外泄露

    • OpenAI 最新模型瞄准两大能力突围
    • 多处踪迹被扒，引发社区关注

  4. GPT-5 更新至 5.3-Codex

    • 最强大、全面、Agent 能力增强
    • 推荐场景：复杂任务、编程

  5. Claude Opus/Sonnet 4.6 发布

    • 安全、写作强、长任务处理能力提升
    • 推荐场景：长文本、分析、编程

  6. 谷歌 Gemini 升级至 3.1 Pro 版本

    • 多模态能力进一步提升

  7. 通义千问 Qwen 2.5 发布

    • 中文能力强大，部分版本免费开放

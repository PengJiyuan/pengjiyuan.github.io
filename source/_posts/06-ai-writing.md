## AI 写作的应用场景

- 📝 写博客文章
- 📧 写商务邮件
- 📄 写简历/求职信
- 📚 写学习笔记
- 📝 写社交媒体文案

---

## 基础写作 Prompt

### 写文章
```
"帮我写一篇 [主题] 的文章，
要求：
- 字数 [xxx]
- 风格 [正式/轻松/幽默]
- 受众是 [谁]"
```

### 写邮件
```
"帮我写一封 [场景] 的邮件，
要求：
- 语气 [礼貌/正式/友好]
- 收件人是 [谁]"
```

### 写简历
```
"帮我写一份 [岗位] 的简历，
突出以下优势：
1. [优势1]
2. [优势2]
3. [优势3]"
```

---

## 进阶技巧

### 1. 给 AI 喂参考内容
```
"根据下面内容，帮我写一篇总结：
[粘贴内容]"
```

### 2. 让 AI 模仿风格
```
"模仿 [作者名/公众号名] 的风格，写一篇关于 [主题] 的文章"
```

### 3. 分段生成
```
"先帮我列出文章大纲，
然后我们逐部分完善"
```

---

## 注意事项

⚠️ **AI 是辅助，不是替代**
- AI 写的文章要自己修改
- 加入自己的观点和经历
- 检查事实准确性

⚠️ **避免重复**
- 不要完全照搬 AI 生成的
- 加入个人特色

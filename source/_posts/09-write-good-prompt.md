## 什么是 Prompt？

**Prompt = 提示词 = 你对 AI 说的话**

你跟 AI 说得越清楚，AI 回答得越好！

---

## Prompt 核心原则

### 1️⃣ 明确具体

❌ 差："写一篇关于狗的文章"
✅ 好："写一篇 500 字的科普文章，介绍柯基犬的性格特点和饲养注意事项"

### 2️⃣ 给出背景

❌ 差："帮我写代码"
✅ 好："我正在学习 Python，需要一个计算器程序，要求有加减乘除四个功能"

### 3️⃣ 指定格式

❌ 差："介绍一下北京"
✅ 好："用表格形式介绍北京，包括人口、面积、著名景点、气候特点"

### 4️⃣ 分步提问

❌ 差："帮我写一个网站"
✅ 好：分步骤
1. "帮我设计一个个人博客的架构"
2. "用 HTML 写首页结构"
3. "用 CSS 写样式"

## Prompt 模板

### 📝 文章写作
```
请帮我写一篇关于 [主题] 的 [类型]，要求：
- 字数：[数字]
- 风格：[正式/轻松/幽默]
- 受众：[是谁]
- 要包含：[要点1]、[要点2]
```

### 💻 代码生成
```
用 [编程语言] 写一个 [功能]，
要求：
1. [要求1]
2. [要求2]
主要用于 [使用场景]
```

### 📊 学习解释
```
用通俗易懂的语言解释 [概念]，
类比 [生活中的例子]，
并给出 [数字] 个例子
```

## 进阶技巧

### 指定角色
"你是一位资深程序员，帮我审查这段代码..."

### Few-shot 示例
"下面是小说的开头，请续写：
'那天天气很好，...'"

### 迭代优化
如果第一次回答不满意，可以：
- "刚才的回答中，关于 XX 部分可以详细说说吗？"
- "换一个角度，重写一下"
- "再简洁一些"

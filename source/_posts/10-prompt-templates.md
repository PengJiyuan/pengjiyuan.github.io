## 📝 内容创作

### 写文章
```
请帮我写一篇关于 [主题] 的文章，
要求：
- 字数：[xxx]
- 风格：[正式/轻松/幽默]
- 受众：[谁]
- 要包含：[要点1、2、3]
```

### 写文案
```
你是一个 [角色]，
帮我写一个 [产品/服务] 的推广文案，
特点：
1. [特点1]
2. [特点2]
目标用户：[谁]
```

### 写故事
```
写一个 [类型] 故事，
主题是 [主题]，
主角是 [人物设定]，
情节要 [要求]，
字数 [xxx]
```

## 💻 编程相关

### 写代码
```
用 [语言] 写一个 [功能]，
要求：
1. [要求1]
2. [要求2]
主要用于 [场景]
```

### 解释代码
```
帮我解释这段代码是做什么的：
[粘贴代码]
```

### 修 bug
```
这段代码报错了：
[粘贴错误信息]
[粘贴代码]
帮我找出问题并修复
```

## 📚 学习相关

### 解释概念
```
用通俗易懂的语言解释 [概念]，
类比 [生活中的例子]，
并给出 [数字] 个例子
```

### 做笔记
```
帮我把以下内容整理成笔记：
[粘贴内容]
要求：
- 提取重点
- 用列表形式
- 加入自己的理解
```

### 出题练习
```
出 [数量] 道关于 [主题] 的选择题，
要包含答案和解析
```

## 📊 办公相关

### 写邮件
```
写一封关于 [主题] 的邮件，
收件人：[谁]
场景：[什么情况]
语气：[什么风格]
```

### 做 PPT 大纲
```
帮我设计一个关于 [主题] 的 PPT 大纲，
要求：
- [数量] 页
- 包含：[要点]
- 受众：[谁]
```

### 数据分析
```
分析以下数据：
[粘贴数据]
给出会计和可视化建议
```

## 使用技巧

### 迭代优化
1. 先让 AI 生成初稿
2. 提出具体修改意见
3. 重复直到满意

### 给角色
"你是一位 [职业]，帮我..."

### 给格式
"用表格/列表/代码块的形式输出"

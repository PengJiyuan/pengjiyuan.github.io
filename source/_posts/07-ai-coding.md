## 为什么用 AI 写代码？

- ⚡ **速度快** - 几秒钟生成代码
- 📚 **学习帮手** - 解释代码教你看懂
- 🐛 **调试专家** - 帮你找 bug 修 bug
- 💡 **灵感来源** - 给你提供思路

---

## 基础用法

### 1. 让 AI 写代码

```
"用 Python 写一个计算器，要有加减乘除功能"
```

### 2. 让 AI 解释代码

```
"帮我解释这段代码是做什么的：
[粘贴代码]"
```

### 3. 让 AI 修复 bug

```
"这段代码报错了，帮我看看哪里有问题：
[粘贴代码和错误信息]"
```

### 4. 让 AI 优化代码

```
"帮我优化这段代码，让它运行更快：
[粘贴代码]"
```

## 实战示例

### 场景 1：做一个网页

**Prompt：**
> "用 HTML + CSS 做一个个人简介页面，包含姓名、头像、自我介绍和联系方式"

### 场景 2：处理数据

**Prompt：**
> "用 Python 读取一个 CSV 文件，计算每列的平均值，输出到新文件"

### 场景 3：自动化脚本

**Prompt：**
> "写一个 Python 脚本，每天早上 9 点自动给我发送天气预报邮件"

## 最佳实践

### ✅ 要明确需求
❌ "写个程序"
✅ "用 Python 写一个计算器，要有 GUI 界面"

### ✅ 说明使用场景
❌ "写个排序算法"
✅ "写一个快速排序算法，用来处理 100 万条数据"

### ✅ 让 AI 加注释
**Prompt：**
> "写完代码后，加上详细的中文注释，解释每一步在做什么"

### ✅ 让他分步骤
**Prompt：**
> "分步骤实现这个功能，每一步单独说明"

## 注意事项

⚠️ **AI 写的代码不一定完美**
- 可能有 bug
- 可能不是最优解
- 重要代码要自己检查

⚠️ **安全意识**
- 不要把敏感信息发给 AI
- 生成的代码要审查

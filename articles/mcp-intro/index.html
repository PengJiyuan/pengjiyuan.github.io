MCP 定义#

MCP = Model Context Protocol = 模型上下文协议

由 Anthropic（Claude 母公司）于 2024 年 11 月发布

一句话解释：让 AI 能"动手"做事的通用标准

MCP 的核心概念#

1. MCP Host（宿主）#

  • AI 应用本身
  • 如：Claude Desktop、Cursor

2. MCP Server（服务器）#

  • 连接 AI 和外部工具的"翻译器"
  • 每个工具对应一个 MCP Server

3. Resources（资源）#

  • AI 可以读取的数据
  • 如：文件、数据库内容

4. Tools（工具）#

  • AI 可以执行的操作
  • 如：搜索、发送邮件、读写文件

5. Prompts（提示词）#

  • 预设的提示模板
